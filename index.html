Your Blends

How infinity bottles work

An infinity bottle is a blending bottle you keep topping off with leftover pours instead of letting the last ounce of every bottle go to waste. It never truly empties — every addition changes the blend a little, so no batch tastes quite the same twice.

  • Start with any clean empty bottle — the size doesn't matter.
  • Pour in what's left of a bottle you're finishing, rather than dumping in full pours.
  • Keep proof roughly in the same range so one addition doesn't overpower the blend.
  • Log every addition — it's the only way to reverse-engineer why a batch turned out great.
  • There's no "finishing" it. Keep adding, keep tasting, keep logging.
